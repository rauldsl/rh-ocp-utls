Edition Deep Dives

Explore each individual edition's target use-cases, built-in features, and hardware parameters.

Virtualization Only Focus

OpenShift Virtualization Engine

OVE
Virtualization Capability: VMs Only

Core Purpose

OVE is designed to run and manage virtual machine workloads only and does not include the ability to run OpenShift containers directly on the bare metal.

Capabilities Included

  • Admin Web Console & Core Kubernetes Infrastructure
  • Full Virtual machine-based workload hosting (KVM VMs) only
  • Compliance & File Integrity Operators

RHEL Guest Entitlements

OpenShift Virtualization Engine (OVE) does not include unlimited Red Hat Enterprise Linux (RHEL) guest entitlements.

Core Enterprise

OpenShift Kubernetes Engine

OKE
Virtualization Capability: Included Standard

Core Purpose

Builds the foundational enterprise-grade Kubernetes footprint. Delivers smart installations, core platform metrics/forwarding, ingress, storage, network segmentation, and over-the-air updates. Fully supports running VMs alongside containers natively.

Capabilities Included

  • Complete physical & virtual container hosting
  • First-class virtual machine hosting side-by-side with containers
  • Platform logging & User workload monitoring

RHEL Guest Entitlements

Includes unlimited Red Hat Enterprise Linux (RHEL) Guest entitlements on physical hosts/bare metal footprint.

Developer-Focused

OpenShift Container Platform

OCP
Virtualization Capability: Included Standard

Core Purpose

A comprehensive application platform that wraps OKE with complex application, pipeline, and developer services. Accelerates development speed, streamlines pipeline management, and enables developer teams to treat VMs and containers as code inside GitOps/CI-CD workflows.

Capabilities Included

  • Everything in OpenShift Kubernetes Engine (OKE)
  • Serverless, Pipelines, Service Mesh, & OpenShift GitOps
  • Manage VM lifecycle configurations within the unified dev catalog

RHEL Guest Entitlements

Includes unlimited Red Hat Enterprise Linux (RHEL) Guest entitlements on physical hosts/bare metal footprint.

Security & Multi-Cluster

OpenShift Platform Plus

OPP
Virtualization Capability: Included Standard

Core Purpose

A complete operational engine designed for massive multicloud scaling. Layered directly over OCP to deliver advanced compliance policies, multicluster observability, centralized image management, and block storage. Offers enterprise multicluster governance of VMs.

Capabilities Included

  • Everything in OpenShift Container Platform (OCP)
  • Advanced Cluster Management (ACM) & Advanced Cluster Security (ACS)
  • Global-scale storage, data backups, and security policies for hybrid VMs

RHEL Guest Entitlements

Includes unlimited Red Hat Enterprise Linux (RHEL) Guest entitlements on physical hosts/bare metal footprint.

Managed Cloud Services Options

Sustained by Red Hat site reliability engineering (SRE) directly within public clouds, optimizing operating expenses and minimizing infrastructure management overhead.

Self-Managed Deployments

With self-managed editions (OVE, OKE, OCP, OPP), you deploy, secure, configure, and upgrade the cluster on physical nodes, private virtual clouds, or custom hyperscaler VPCs. Your operations team is responsible for cluster health, backups, and underlying virtualization maintenance.

Your Management Scope (Full Stack):

⚠ Hardware, Virtualization Layer, & Guest OS
⚠ Control Plane Installation & Smart Updates
⚠ Core Cluster Metrics, Logging, Security Controls
✓ Developer Workspaces, Applications, & App Data

Fully Managed Cloud Services

With managed cloud configurations, Red Hat SREs (jointly with partner cloud providers) assume operational support of the control planes, master nodes, and infrastructure layers. You benefit from proactive 24x7 monitoring, automatic security patches, and rapid cluster deployment.

Platform Management Scope:

✓ Managed by Red Hat/Partner SRE: Hardware Provisioning & Bare-Metal Nodes
✓ Managed by Red Hat/Partner SRE: Control Plane Upgrades, Patches, & 99.95% SLA
✓ Managed by Red Hat/Partner SRE: Proactive 24x7 Incident Management
★ Your Scope: Simply deploy your VM workloads, containers, and code
